# Postman

All Postman collections and environments live in this directory.

- **Collections:** Place `.postman_collection.json` files here (e.g. `DietWatcher-API.postman_collection.json`).
- **Environments:** Place `.postman_environment.json` or `environment.json` here.
- **Usage:** Import the collection and environment in Postman; use the environment for base URL and API keys (never commit secrets).

Do not scatter Postman JSON files elsewhere in the project.
